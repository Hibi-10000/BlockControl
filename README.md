# BlockControl
Bukkit(Spigot,Paper) 1.16.5~  
  
ブロックの設置を制限できます。  
使い方は/bc helpを見ればだいたい分かる。  
ワールドごとに制限できるように後でします。  
  
config.yml  
BanBlock:  
  \- TNT  #置けないようにするブロック名(大文字)  
  \- LAVA  
WhiteWorld:  
  \- whiteworld #BanBlockを置けるようにするワールド名  
  
不具合があったら報告していただけると助かります。
